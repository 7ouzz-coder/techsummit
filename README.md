# TechSummit 2025 - Landing Page

Página web para una conferencia de tecnología hecha con Bootstrap.

[Repositorio de Github](https://github.com/7ouzz-coder/techsummit)

## ¿Qué es este proyecto?

Es una landing page de una sola página para promocionar un evento de tecnología. Tiene información del evento, oradores, agenda y formulario de registro.

## Tecnologías que usé

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Archivos del proyecto

- `index.html` - La página principal
- `css/styles.css` - Los estilos personalizados
- `js/script.js` - El código JavaScript
- `assets/` - Las imágenes del evento

## Cómo abrir el proyecto

1. Descarga o clona el proyecto
2. Abre el archivo `index.html` en tu navegador
3. Listo

## Qué hace el JavaScript

- Valida el formulario cuando lo envías
- Muestra mensajes de error si hay campos incorrectos
- Muestra mensaje de éxito cuando todo está bien
- Hace scroll suave cuando haces clic en el menú

## Componentes de Bootstrap que usé

- Navbar (menú de navegación)
- Cards (tarjetas de información)
- Buttons (botones)
- Forms (formulario)
- Modal (ventanas emergentes)
- Accordion (acordeón de la agenda)
- List Group (listas)
- Badges (etiquetas de horarios)

## Responsive

La página se ve bien en:
- Celulares
- Tablets
- Computadoras
